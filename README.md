QuoteNova 📖✨

QuoteNova is a modern Android application built with Kotlin and Jetpack Compose that delivers inspirational quotes through a clean and minimal user interface.
The application displays a random quote each time the user opens the app or requests a new one. Users can also share their favorite quotes with friends using the built-in sharing feature.

📱 Features

 Random quote generation,
 Display quote text and author name,
 Generate a new quote with a single tap,
 Share quotes with other applications,
 Smooth quote transition animations,
 Modern Material 3 user interface,
 Responsive and user-friendly design
