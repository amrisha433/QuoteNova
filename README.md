# 📖 QuoteNova

QuoteNova is a modern Android application built with **Kotlin** and **Jetpack Compose**
that delivers inspirational and motivational quotes through a clean, elegant, and user-friendly interface.

The app provides a distraction-free experience where users can discover inspiring quotes,
switch between light and dark themes, and share their favorite quotes with friends and family
using Android's built-in sharing functionality.

---

## ✨ Features

* 🎲 Random quote generation
* 📝 Display quote text with author information
* 🔄 Generate a new quote with a single tap
* 📤 Share quotes with other applications
* 🌙 Dark Mode & ☀️ Light Mode support
* ✨ Smooth quote transition animations
* 🎨 Modern Material 3 design
* 📱 Responsive and user-friendly interface
* ⚡ Built entirely with Jetpack Compose
